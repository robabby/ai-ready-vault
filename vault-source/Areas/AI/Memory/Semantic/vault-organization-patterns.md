# Vault Organization Patterns

## PARA Method

- **Projects**: Active work with deadlines
- **Areas**: Ongoing responsibilities
- **Resources**: Reference material
- **Archive**: Completed or inactive items

## Memory Types

| Type | Purpose | Example |
|------|---------|---------|
| Episodic | Specific events | Session logs, meetings |
| Semantic | Facts & knowledge | This note |
| Procedural | How-to guides | Workflows, processes |
| Strategic | Goals & decisions | Principles, plans |

## Best Practices

1. **Capture quickly** - Use Inbox for fast capture
2. **Process regularly** - Move items to proper locations
3. **Link generously** - Connect related notes
4. **Review periodically** - Consolidate and archive
