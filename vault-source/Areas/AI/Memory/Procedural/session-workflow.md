# Session Workflow

## Before Starting

1. Review recent session logs
2. Check active projects for context
3. Load relevant memories

## During Session

1. State the goal clearly
2. Work collaboratively
3. Ask clarifying questions
4. Document decisions as you go

## After Session

1. Create session log from template
2. Identify memories to create:
   - New learnings (semantic)
   - Significant events (episodic)
   - New processes (procedural)
   - Goal/decision changes (strategic)
3. Update related notes with links
4. Archive completed tasks

## Commands

```
/hydrate - Load context from memories
/reflect - End session and consolidate
/remember - Store new memory
/recall - Search existing memories
```
